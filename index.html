内网穿透

”使公网可以访问处于NAT网络下或多级路由下的服务”

下载

下载与安装

ngrok 易于安装，常用平台都有二进制包，下载即可运行。

下载 ngrok

Linux ARM ARM
Linux 64位   32位
Windows 64位   32位
Mac OS X 64位   32位

配置

创建配置文件 ngrok.cfg

server_addr: ngrok.wkfg.org:4443
trust_host_root_certs: false

运行

命令行启动，成功后即可通过 http://demo.wkfg.ngrok.org 来访问本机 8080 端口的服务

ngrok.exe -subdomain=demo -config=ngrok.cfg -log=ngrok.log 8080

抓包

访问 127.0.0.1:4040 即可看到请求的详细信息
